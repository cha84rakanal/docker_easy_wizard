# docker_easy_wizard

Docker runコマンドをWizard形式で構築できるReact + TypeScript + MUIアプリです。

## 開発

```bash
yarn install
yarn dev
```

## ビルド

```bash
yarn build
yarn preview
```

## 今後のタスク

優先度高
- 永続化: 作成コマンドの保存と一覧復元
- 編集/複製/削除: 既存エントリの再編集と管理
- コピー導線: クリップボードコピーとターミナル向け整形
- 追加オプション拡張: env/network/restartなど
- 入力バリデーション: ポート/パス/必須チェック
- Docker Hubフォールバック: キャッシュと失敗時の表示
